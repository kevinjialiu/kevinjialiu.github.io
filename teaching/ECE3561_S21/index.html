Course Topics

    • Sequential circuit analysis.
    • Timing issues (maximum clock frequency, setup time, hold time, clock skew).
    • Sequential circuit design.
    • Analysis of sequential circuits with logic building blocks and system controller.
    • System controller design.
    • More timing issues: asynchronous inputs, glitch-free outputs.
    • Design with counters, shift registers multiplexers, comparators, decoders, adders, PLDs, and FPGAs.
    • Design technology: VHDL, Xilinx.

Course Materials

    • C.H. Roth and L.L. Kinney, "Fundamentals of Logic Design," 7th edition, Cengage Learning, 2014.
    • Reference Book: John F. Wakerly, "Digital Design: Principles and Practices," 4th edition, Pearson, 2005.

Schedule

Here is an estimated class schedule, which is subject to change depending on lecture progress and/or class interests. Please check for latest adjustments.

Week Topics
1 Introduction to Sequential Components; SR Latch; Gated D Latch
2 Edge Triggered D Flip Flop; SR, JK, T Flip Flops
3 VHDL for Combinational Circuits; Basic Sequential Circuit Analysis; Timing Analysis
4 Decomposition of State Machines; Analysis of Sequential Circuits with MSI Components
5 System Controller Analysis; Applications of commonly used Sequential MSI Components
6 Sequential Circuit Design Overview; 9-Step Approach; Ad Hoc Design; Component-Centric Design
7 Midterm; VHDL for Sequential Circuits
8 System Controller Design; Advanced Synchronous Circuit Design Approaches
9 Construction of State Diagrams; Formal Methods of State Minimization
10 Formal Methods of State Minimization (continued); State Assignment Procedures
11 Iterative Circuits; Adders and Binary Serial Adder
12 Asynchronous Inputs; Output Glitches; Memory
13 Programmable Logic Devices and Complex Programmable Logic Devices
14 Midterm; Field Programmable Gate Arrays, Shannon's Expansion Theorem; Review for Final
